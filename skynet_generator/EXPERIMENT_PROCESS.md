# Skynet Experiment Process

This project ports the HallTime architecture into sci-fi villain scenario generation. It keeps the same broad loop:

1. structured scaffold generation
2. optional LM expansion by default
3. corpus indexing
4. retrieval-backed editorial refinement
5. recurrence pressure on later generations

The current goal is not final gameplay. It is proving that the same symbolic digestion loop can work for prestige systems paranoia, rogue AI melodrama, black-site horror, and optimization-drift villainy.
